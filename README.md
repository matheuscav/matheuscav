## 🎬 Projeto em Destaque: `whatsapp-financeiro`

> Bot de controle financeiro do casal, direto pelo WhatsApp.

**Stack:** Node.js · Baileys · MariaDB · Power BI

```
Mensagem no WhatsApp
   "gasto 45 credito alimentacao mercado"
        │
        ▼
   [ parser.js ]  →  interpreta tipo, valor, categoria
        │
        ▼
   [ db.js ]      →  grava em financas_casal.lancamentos
        │
        ▼
   [ Power BI ]   →  dashboard atualizado
```

📍 Status: código e schema prontos → deploy em Oracle Cloud Always Free (Ubuntu 22.04 ARM), com hardening completo (SSH key-only, UFW, Fail2ban, backups criptografados).
